# Ethical Boundaries

The hard lines `marketing-psychology` does not cross. Applies to every phase,
every surface, every principle. Surfacing any of these → stop and re-design.

## Dark patterns (never use)

- **Fabricated stats.** Invented metrics, fake testimonials, inflated user
  counts. If you cannot cite a real source, omit the figure.
- **Manufactured urgency.** Countdown timers that reset on reload, "only 1
  left" when stock is plentiful, fake scarcity. Real scarcity only.
- **Confirm-shaming.** Opt-out copy that insults the user ("No thanks, I'd
  rather keep missing calls"). Always offer a neutral opt-out.
- **Roach motel.** Easy to sign up, hard to cancel. Cancellation must be as
  easy as sign-up.
- **Bait-and-switch.** Advertise one price/feature, deliver another.
- **Privacy dark patterns.** "Accept all cookies" in bright color, "Reject"
  hidden behind two menus. Symmetric consent UI.

## Addiction design (never use)

- Infinite scroll without an explicit "stop and exit" affordance.
- Push notifications optimized for habit-formation rather than user value.
- Variable reward schedules used to drive engagement rather than genuine
  utility.
- Any pattern whose primary mechanism is exploiting dopamine response.

## Override product truth (never do)

- Claiming a feature works in a way it does not.
- Describing a tier's limits inaccurately to push upgrades.
- Stating integrations exist when they do not.
- Hiding material limitations (compliance, geography, supported languages).

## The ethical-influence test

Before shipping any copy surfaced by this skill, ask:

1. Would I be comfortable if the user saw exactly how this was constructed?
2. Does this work *because* the user understands it, or *because* they don't?
3. If the user paused and thought carefully, would this still work?

If any answer is "no, this only works if they don't notice," it is manipulation,
not influence. Redesign or drop.
